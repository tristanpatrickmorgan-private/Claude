# Personal Assistant

A simple personal life organiser powered by Claude + GitHub.

Chat with Claude to capture anything — tasks, shopping lists, gym sessions, notes, plans. Everything gets saved here as plain markdown files.

## How to use

Just talk to Claude naturally:

- "Add milk and bread to the shopping list"
- "Log gym: squats 3x8 at 100kg, bench 3x10 at 70kg"
- "Remind me to call the dentist"
- "Note: discussed Portugal for September holiday with Sarah"
- "What's on my to-do list?"

Claude handles the file updates and commits automatically.

## Structure

```
todo.md          — tasks and to-dos
shopping.md      — shopping lists
logs/
  gym.md         — gym and fitness log
  YYYY-MM.md     — general daily notes
notes/
  holidays.md    — trip and holiday planning
  (anything else you need)
```

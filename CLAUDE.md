# Instructions for Claude

This repo is a personal log and task tracker. When I send tasks:

- To add a log entry: append to `logs/YYYY-MM.md` under a `## YYYY-MM-DD` heading.
  Create the file if it doesn't exist. Newest entries at the bottom.
- To update tasks: edit `todo.md`. Use `- [ ]` for open and `- [x]` for done.
  When completing a task, move it to the `## Done` section with the date.
- Always commit with a clear message. Open a PR; don't push to main directly.
- Keep changes minimal — don't reformat unrelated content.
